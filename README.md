# PEMBANGUNAN SISTEM INFORMASI TIMER, BOOKING & MANAGEMENT PADA PRINCE GAMES BERBASIS WEB ## Prince Game Rental PS adalah sistem berbasis web yang dirancang untuk membantu pengelolaan usaha rental PlayStation secara efisien dan modern. Sistem ini memiliki dua peran utama, yaitu admin dan pelanggan, dengan fitur utama seperti manajemen waktu bermain, booking unit PS, penambahan durasi oleh admin, serta monitoring status setiap unit (PS3, PS4, dan PS5). Selain itu, sistem ini dilengkapi dengan pemesanan makanan, berbagai metode pembayaran (cash, e-wallet, dan transfer), serta pencatatan transaksi yang otomatis masuk ke laporan. Tersedia juga fitur notifikasi/alarm saat waktu bermain habis, serta rekap pendapatan harian. Dengan tampilan yang responsif dan user-friendly, sistem ini dapat diakses melalui desktop maupun perangkat mobile, sehingga memudahkan operasional rental secara real-time dan lebih terstruktur ## Tim Pengembang - [Dinar Jhonata] [Dapa Putra Wiradhika Kismaja] [Muhammad Taufik Septiadika] - Product Owner - [Nurhaida br.hombing] - Scrum Master - [Syabila Alfani] - Developer ## Teknologi - [html, js]
